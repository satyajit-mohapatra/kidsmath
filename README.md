## How to Use

### Local Development

1. **Open the App**: Simply open `index.html` in any modern web browser

```bash
# macOS
open index.html

# or open specific game
open addition.html
```

2. **Select a Game**: Choose from the main menu

3. **Solve Problems**:
   - Look at the visual objects (emojis) representing each number
   - Type your answer in the box
   - Press Enter or click "Check!"

4. **Get Help**: Click "Need a hint?" if stuck

5. **Track Progress**: Watch stars and streak grow!

## Tips for Parents/Teachers

- **Start Easy**: Begin with Level 1 to build confidence
- **Use Visuals**: Encourage counting the emoji objects
- **Short Sessions**: Keep sessions short (10-15 minutes) for hyperactive children
- **Celebrate Success**: Praise correct answers and streaks
- **Use Hints**: Don't hesitate to use hints to prevent frustration
